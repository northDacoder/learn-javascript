Why use JavaScript?

  • Community: JavaScript has one of the largest communities for learning & is used by almost every company in one way or another
  • Flexible: It is extremely flexible in what it accepts, which can be both a benefit and a hindrance.
  • Speed: JavaScript is fast & client-side, which reduces unneccessary trips to the server
  • Libraries: JavaScript has libraries for almost anything you can think of

What does it do?

While HTML defines the content, and CSS is is used for styling and layouts, JavaScript allows the user to interact with your web page. Ironically, coming from learning Python, JavaScript probably seems rather rigid and ugly, with all of its semi-colons and curly braces, but compared to languages like Objective C or Java, it is actually quite concise and accommodating.

Comments


Comments are statements that will be ignored by the JavaScript interpreter. Comments are used to describe what your code does (or what you think it does when you wrote it!).


Comments make it easier for developers to start contributing to a project they didnt start without spending too much time trying to figure out on their own what the code does.

Single-line Comments



// Define a single line comment that will be ignored by the interpreter
var sayHello - "Hello, welcome to my home page";
var sayGoodbye = "Thank you!";

What is a variable?

Variables are symbolic names for values in your application. Variable names names must start with a letter, an underscore (_), or a dollar-sign ($). You can also use numbers (0-9) as long as it is after the first character in the variable name


var firstName = "Jimmy";
var $lastName = "Johns";
var _favoriteSub = "Vito";



var 5vito = "Vito Sandwich";
var *important = "Add chips";
